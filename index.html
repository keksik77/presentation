React Native

What it is?

  • This is a framework for developing cross-platform applications using js and React
  • Created by Facebook
  • Appeared at the beginning of the 2015 year
  • React based
  • Under the hood does not use webView/UIWebView and HTML technology(Dom)
  • Uses js API on top of native components

Inside look

  • No HTML, as a replacement - components of the JSX platform, which we will discuss further
  • No CSS styles, replacement - CSS-like polyphiles
  • No DOM API

Architecture React Native

  • The main stream - handles the display of user interface elements and user gestures.
  • JavaScript stream - it executes JavaScript code in a separate engine, determines HOW the application works.

For communication between the two streams meets the so-called bridge - a core React Native.

To start

  • Expo
  • React Native CLI

Expo

Expo - a set of tools, libraries and services that allow you to quickly start.

What about cross-platform?

There are 2 ways to split code across platforms:

  • Using the Platform Module
  • Using platform-specific file extension

Platform.OS


							import {Platform, StyleSheet} from 'react-native';

							const styles = StyleSheet.create({
								height: Platform.OS === 'ios' ? 200 : 100,
							});

Platform.select


							import {Platform, StyleSheet} from 'react-native';

							const Component = Platform.select({
							  ios: () => require('ComponentIOS'),
							  android: () => require('ComponentAndroid'),
							})();

							<Component />;

							const styles = StyleSheet.create({
							  flex: 1,
							  ...Platform.select({
							    ios: {
							      backgroundColor: 'red',
								},
							    android: {
								  backgroundColor: 'blue',
							    },
							  }),
							});

Navigation

React Navigation

  • cross-platform
  • native UI for each platform, you don’t have to worry about smooth transitions, etc.
  • . basic types of transitions:
    / createStackNavigator
    / createSwitchNavigator
    / createDrawerNavigator
    / createBottomTabNavigator ...

React Navigation example


							const HomeStack = createStackNavigator({
								Home: {screen: HomeScreen },
								Details: { screen: DetailsScreen },
							});
							const SettingsScreen = createStackNavigator({
								Settings: { screen: SettingsScreen },
								Details: { screen: DetailsScreen },
							});

React Navigation example


							class HomeScreen extends React.Component {
							  static navigationOptions = {title: 'Home'};
							  render() {
							    return (
							      <View style={style.container}>
								    <Text style={styles.welcome}>Hello world!!!</Text>
								    <Button title="Go to Details" onPress={() =>
								      this.props.navigation.navigate('Details')}/>
								  </View>
							    );
							  }
								...
							}
